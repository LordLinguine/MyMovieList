** HOW TO CLONE THE GITHUB REPO **

Open a new Vscode window
Click "Clone Git Repository" in the Middle Section of the Welcome Page
Click "Clone from GitHub"
Click on the correct Github Repo
Save it to a place where you can find it.
When working on the project open this folder.
** HOW TO GET LATEST VERSION OF THE GITHUB REPO **

Restart Vscode
Click on "Source Control" on the left sidebar
Under "Incoming" click on the right arrow to pull and update the code.
** HOW TO RUN THE FRONTEND (Client) **

MUST have Nodejs installed

Open the terminal in Vscode:
Go into the client folder: enter "cd client" into the terminal
Then run "npm install" to install the packages
Then run "npm start" to run the frontend
** HOW TO RUN THE BACKEND & DATABASE (Server) **

MUST have Nodejs installed

Open the terminal in Vscode:
Go into the server folder: enter "cd server" into the terminal
Then run "npm install" to install the packages
Then run "npm run server" to run the server & connect to the database
** HOW TO MAKE CHANGES TO THE GITHUB REPO **

Open the folder you saved with the project and then make your changes.
When you are DONE with your changes:
Go to the left sidebar on Vscode:
Click on "Source Control"
Click the PLUS Icon next to "Changes"
Enter a message in the top text input - Format: "[Frontend/Backend/Database] [Date]: [message] (Keep it short)"
Ex: "Frontend (3/10/24): Finished Home Page UI"
Click on "Commit"
Click on "Sync Changes" to push to the main GitHub Repo
